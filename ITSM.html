Árvore ITSM Interativa - ITIL/COBIT

  • 1. Incidentes

    Descrição: Chamados para restaurar serviços afetados.

    Tipo: Incidente

    Responsável / Grupo: Equipe de Suporte / Infraestrutura

    • Hardware
      • Computadores / Laptops (SLA: 4h crítico / 8h médio / 24h baixo)
      • Impressoras / Scanners (SLA: 8h crítico / 16h médio / 48h baixo)
      • Equipamentos de rede (SLA: 2h crítico / 4h médio / 12h baixo)
      • Outros periféricos (SLA: 8h médio / 24h baixo)
    • Software
      • Sistemas críticos (ERP, CRM, Financeiro) (SLA: 2h crítico / 4h médio / 8h baixo)
      • Aplicativos de produtividade (SLA: 4h médio / 16h baixo)
      • Navegadores / Plugins (SLA: 8h médio / 24h baixo)
    • Acesso e Autenticação
      • Reset de senha (SLA: 1h crítico / 4h médio)
      • Conta bloqueada (SLA: 1h crítico / 2h médio)
      • Permissões incorretas (SLA: 4h médio / 12h baixo)
    • Rede e Conectividade
      • Internet lenta / instável (SLA: 2h crítico / 8h médio)
      • VPN / acesso remoto (SLA: 2h crítico / 4h médio)
      • Wi-Fi indisponível (SLA: 4h crítico / 8h médio)
  • 2. Requisições de Serviço

    Descrição: Solicitações de serviços padrão e provisionamento de recursos.

    Tipo: Requisição

    Responsável / Grupo: Equipe de Suporte / Atendimento ao Usuário

    • Hardware
      • Novo equipamento (SLA: 24h médio / 72h baixo)
      • Periféricos (SLA: 24h médio / 72h baixo)
      • Substituição de equipamento (SLA: 8h crítico / 24h médio)
    • Software
      • Instalação / atualização (SLA: 8h médio / 24h baixo)
      • Solicitação de licenças (SLA: 8h médio / 24h baixo)
      • Configuração de aplicativos (SLA: 8h médio / 24h baixo)
    • Acesso e Permissões
      • Novo usuário / conta (SLA: 4h crítico / 8h médio)
      • Alteração de perfil / função (SLA: 8h médio / 24h baixo)
      • Solicitação de acesso a sistemas (SLA: 8h médio / 24h baixo)
    • Informação e Suporte
      • Orientações de sistemas (SLA: 24h médio / 48h baixo)
      • Treinamento (SLA: 48h médio / 72h baixo)
      • Consultoria técnica (SLA: 24h médio / 48h baixo)
  • 3. Problemas

    Descrição: Análise de causas raiz para prevenir incidentes recorrentes.

    Tipo: Problema

    Responsável / Grupo: Equipe de Infraestrutura / TI

    • Problema recorrente de hardware (SLA: 48h médio / 7 dias baixo)
    • Problema recorrente de software (SLA: 24h crítico / 72h médio)
    • Análise de causa de incidentes críticos (SLA: 8h crítico / 24h médio)
  • 4. Mudanças

    Descrição: Solicitação e implementação de alterações controladas na TI.

    Tipo: Mudança

    Responsável / Grupo: Change Advisory Board / Equipe de TI

    • Atualizações de software / sistemas (SLA: conforme janela de manutenção / 24h médio)
    • Migração de sistemas ou dados (SLA: conforme planejamento / 72h baixo)
    • Alterações de rede / infraestrutura (SLA: 8h crítico / 24h médio)
    • Implementação de novos serviços (SLA: Planejado / 5 dias baixo)
    • Solicitação de aprovação de mudança crítica (SLA: 4h crítico / 24h médio)
  • 5. Gerenciamento de Ativos / Configuração

    Descrição: Controle e documentação de ativos e configuração da TI.

    Tipo: Ativo / Configuração

    Responsável / Grupo: Equipe de Configuração / Inventário

    • Inventário de hardware/software (SLA: 48h médio / 5 dias baixo)
    • Atualização de CMDB (SLA: 48h médio / 7 dias baixo)
    • Auditoria de ativos de TI (SLA: 5 dias baixo)
    • Retirada / substituição de equipamentos (SLA: 8h crítico / 24h médio)
  • 6. Segurança da Informação

    Descrição: Gestão de incidentes e solicitações de segurança da informação.

    Tipo: Segurança

    Responsável / Grupo: Equipe de Segurança da Informação

    • Malware / vírus (SLA: 1h crítico / 4h médio)
    • Phishing / fraude (SLA: 1h crítico / 2h médio)
    • Conta comprometida (SLA: 1h crítico / 4h médio)
    • Solicitação de antivírus / firewall (SLA: 8h médio / 24h baixo)
    • Criptografia de dados (SLA: 24h médio / 72h baixo)
    • Revisão de políticas de acesso (SLA: 5 dias baixo)
  • 7. Governança, Risco e Conformidade

    Descrição: Controle de risco, conformidade e auditoria de TI.

    Tipo: Governança

    Responsável / Grupo: TI / Compliance / Auditoria

    • Solicitação de relatório de conformidade (SLA: 5 dias baixo)
    • Solicitação de auditoria de TI (SLA: 7 dias baixo)
    • Gestão de risco operacional (SLA: 24h médio / 72h baixo)
    • Revisão de processos e controles (SLA: 5 dias baixo)
  • 8. Outros / Geral

    Descrição: Solicitações gerais, feedbacks ou melhorias.

    Tipo: Outro

    Responsável / Grupo: Equipe de Suporte / Administração

    • Sugestões de melhoria (SLA: 7 dias baixo)
    • Feedback do usuário (SLA: 5 dias baixo)
    • Questões administrativas (SLA: 5 dias baixo)
